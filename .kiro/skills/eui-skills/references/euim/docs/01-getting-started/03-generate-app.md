# Generate an eUI app

**Let's create your first eUI application**

1. create a folder : `mkdir my-eui-app`

2. in the folder : `cd my-eui-app`

3. execute the eui-cli command : `eui-cli`

4. check the version installed : 

    ![eui-cli-version](assets/docs/images/eui-cli-01.PNG)

    useful to check the version that you've installed in a past setup f.e.

5. select the type of application skeleton you want to generate

    ![eui-cli-1](assets/docs/images/eui-cli-03.PNG)

    we select the `Angular` default type, we'll see further application types.

6. select one of more options

    ![eui-cli-2](assets/docs/images/eui-cli-02.PNG)

    we don't select any option here, we'll see them further in details

7. choose if you want to install the dependencies (default: Yes) 

    ![eui-cli-3](assets/docs/images/eui-cli-04.PNG)

your application is being generated, after installing the dependencies (this might take a few minutes),
it'll be served on http://localhost:4200, the default Angular CLI local server.

![eui-cli-4](assets/docs/images/eui-cli-05.PNG)

In the end the default eUI application will render : 

![eui-cli-5](assets/docs/images/eui-cli-06.PNG)
