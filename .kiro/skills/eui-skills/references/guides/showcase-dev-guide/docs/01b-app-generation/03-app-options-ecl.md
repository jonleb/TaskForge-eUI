# Generate an app - default desktop eUI Angular - eUI ECL

## Prerequisites

Follow the **[prerequisites](./docs/01-getting-started/01-prerequisites)** section


## Generation steps

**Let's create your first eUI ECL application**

1. Create a folder : `mkdir my-ecl-app`

2. In the folder : `cd my-ecl-app`

3. Execute the eui-cli command : `eui-cli`

4. Select the type of application skeleton you want to generate

    ![generate-ecl-1](assets/docs/01b-app-generation/gen-app-ecl-1.png)

    We select the `Angular` default type, we'll see in next chapters other application types.

5. Select one of more options

    ![generate-ecl-2](assets/docs/01b-app-generation/gen-app-ecl-2.png)

    Use arrows up and down arrows to navigate here, select ECL application (using SPACE).

6. Select either ECL-EC theme or ECL-EU theme

    ![generate-ecl-2](assets/docs/01b-app-generation/gen-app-ecl-3.png)

    Use arrows up and down arrows to navigate here, select ECL application (using SPACE).

    ![generate-ecl-2](assets/docs/01b-app-generation/gen-app-ecl-3b.png)


7. Choose if you want to install the dependencies 

    ![generate-ecl-4](assets/docs/01b-app-generation/gen-app-ecl-4.png)

Your application is being generated, after installing the dependencies (this might take a few minutes),
it'll be served on http://localhost:4200, the default Angular CLI local server.
