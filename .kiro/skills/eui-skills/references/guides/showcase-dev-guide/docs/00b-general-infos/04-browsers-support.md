# Browser Compatibility

## Overview of Browser Support

The eUI Library framework, accessible at [eUI Library Framework](https://eui.ecdevops.eu/), aligns its browser compatibility with that of Angular, as detailed on [Angular's Browser Support Page](https://angular.io/guide/browser-support). This alignment ensures that eUI Library maintains up-to-date compatibility with modern web browsers.

## Supported Browser Versions

In accordance with Angular's support policy, eUI Library is designed to function seamlessly with the most recent version and the two preceding versions of the browsers listed on Angular's support page. This approach ensures that users accessing the eUI Library through a range of popular browsers receive consistent and reliable performance.

## Testing and Validation

All browsers listed as supported by Angular have undergone thorough testing to verify their compatibility with the eUI Library framework. This testing process is integral to maintaining the high standards of functionality and user experience that eUI Library aims to provide.
