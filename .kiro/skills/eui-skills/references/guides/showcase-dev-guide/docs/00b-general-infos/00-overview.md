# eUI Platform

## Overview
The eUI platform is a comprehensive suite of tools and services designed to streamline the development process of web applications. This platform provides a foundational starting point to accelerate the application development initiation phase.

### Key Focus Areas

1. **Adherence to DIGIT UX Guidelines:** Implementation and rationalization of DIGIT UX and UX CC Design [Guidelines](https://webgate.ec.europa.eu/fpfis/wikis/display/Europa/User+experience), ensuring a consistent user experience across web applications.
2. **Integration of eUI Design System:** Utilizing Figma, aligned with the DG COMM [Europa Component Library](https://ec.europa.eu/component-library/) (ECL) standards, ensuring synchronization from the 15.x release onwards.

## Platform Components

- **eUI Roadmap and Release Cycle:** Documentation on release schedules and future platform developments.
- **UI Angular Components for Internal EC Web Applications:** A range of data-driven components following a common design system, from atomic components to fully-realized layout components based on the ECL framework from DG COMM.
- **UI Angular Components for Public Europa Applications:** Angular components encapsulating the ECL library for streamlined integration of standard HTML/CSS/JS elements.
- **eUI CLI Application Generator:** Provides pre-configured application skeletons, incorporating best practices for rapid development in various frameworks including front-end, Spring Boot, Maven, JEE, and PHP Symfony. Supports ECL options.
- **DevOps Tooling:** Encompasses build tools for Angular-based web applications and facilitates CI processes. Supports large-scale, modular Angular application development with a micro-frontends approach.
- **eUI Mobile (Ionic Angular):** Extends the eUI design system to mobile development, promoting standardization in EC mobile application development.
- **eUI Test Automation Framework (eUI TAF):** Developed with the MyWorkplace/Compass Corporate QA team, offering default component objects for eUI components for end-to-end automation testing. Includes support for transition from Protractor to WebDriver.io in the @eui/automation package. For more information or assistance, contact the [DIGIT.A.4 QA team](mailto:digit-eui-support@ec.europa.eu).
