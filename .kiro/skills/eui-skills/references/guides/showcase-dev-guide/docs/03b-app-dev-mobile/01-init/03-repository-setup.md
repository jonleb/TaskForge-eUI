# App development - Step 1 - Initialization

## Repository setup (optional)

Let's configure our repository and secure the develop and master branches.  

This will force developers of your team to always create a feature branch from develop and send their pull request to it once their work is ready to be reviewed and merged.

We'll also set the deletion prevention of those branches.
