# Getting started - overview

You can find more details specific to your configuration in the 
**[prerequisites](./docs/01-getting-started/01-prerequisites)**
section

## Quick start guide

1. **NodeJS Installation**: 
    
    Ensure you have NodeJS version greater than "^20.19.0 || ^22.12.0 || >=24.0.0", Alternatively, install [**nvm**](https://github.com/nvm-sh/nvm) (Node Version Manager).

2. **Yarn Global Installation**:

    Execute `npm i -g yarn` to install Yarn globally on your system.

3. **eUI CLI Global Installation**:

    Run `npm i -g @eui/cli` for global installation of the eUI CLI.

4. **Folder Creation and Navigation**:

    Create a new folder for your project with `md my-eui-app`, and then change directory to it using `cd my-eui-app`.

5. **Execute eui-cli**:

    Start the eui-cli interface by entering `eui-cli` in the command line.

6. **Application Skeleton Selection**:

    When prompted by eui-cli, select your preferred application skeleton type.

7. **Dependencies Installation**:

    Install any required dependencies if asked during the setup process.

8. **Browser Launch with eUI App**:

    Upon completion, your browser will automatically open, displaying the latest eUI application you've set up.
