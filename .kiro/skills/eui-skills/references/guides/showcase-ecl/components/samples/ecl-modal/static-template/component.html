Static inline modals are rendered directly on page and their visibility can be controlled using variety of ways described here.

Controlling modal with directives

Using set of directives to control opening and closing modal

eclModalClose
Closes the modal when host element is clicked. Assigning optional value to this property allows to read it back in modalClose event as eclModalData. The host element must be inside ecl modal component.
eclModalRole
Provides close intention (like cancel, confirm etc). Value of this property allows to read it back in modalClose The host element must be inside ecl modal component.
eclModalTriggerFor
Opens the modal when host element is clicked. Assign modal reference Id to control its visibility.
