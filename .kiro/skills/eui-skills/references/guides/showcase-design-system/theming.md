# 05-theming

## Overview

<p class="eui-u-text-paragraph">
    A <strong>Theme</strong> can be defined as a named set of unique values (tokens) assigned to universal variables (CSS variables) which will affect component's styling.
</p>

<p class="eui-u-text-paragraph">
    When changing aspects of the UI, such as color and typography, <strong>CSS custom properties</strong>, also called CSS variables, 
    allows to apply your design vision throughout the user experience through themes.
    <strong>Themes</strong> allow easy switching between design and code workflows by providing specific values for all component's styles properties.
    By customizing these values, we create a new theme which can be applied to the global application and components.
</p>
