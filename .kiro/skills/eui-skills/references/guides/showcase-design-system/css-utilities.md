# 04-css-utilities

## Overview

<eui-showcase-doc-page id="css-utilities" label="CSS utilities">

    <docPageSections>
        <eui-showcase-doc-section id="Overview">

            <p class="eui-u-text-paragraph">eUI provides various <strong>CSS utility classes</strong> which can be directly used in your Html views to quickly customize the page layout of your elements.</p>

            <h4 class="eui-u-f-bold eui-u-color-info eui-u-mb-m">Usage</h4>
            CSS utilities :<br>
            <div class="eui-u-ml-m">
                <ul>
                    <li>can be used via the component's style <code class="eui-u-text-code">class</code> property</li>
                    <li>provides fast and easy styling of your components and compositions <u>without any extra CSS</u></li>
                    <li>applies CSS style classes which allows to customize a component based on its type class, sizes, states, extra shapes, animations and more</li>
                    <li>allows combining multiple CSS utilities for custom compositions.</li>
                </ul>
            </div>

            <eui-alert class="eui-u-mt-l">
                <eui-alert-title>Important remarks</eui-alert-title>

                It is <strong>highly recommended</strong> to first use the eUI utility style classes instead of creating any extra custom CSS.
                <p class="eui-u-text-paragraph">Advantages over extra custom CSS is that <strong>maintenance</strong> and <strong>compatibility</strong> though various eUI versions are internally <strong>handled and managed by eUI</strong>.</p>
                Also eUI utility classes are based on <strong>Design System tokens</strong> which are the reference in terms of style guide and recommendations.
            </eui-alert>
        </eui-showcase-doc-section>

    </docPageSections>
</eui-showcase-doc-page>
