Usage recommendation Always wrap the input + label within a Flexbox div container by applying the any eUI Flex utility class :
  • eui-u-flex and eui-u-inline-flex: checkboxes are centered aligned
  • eui-u-display-flex: checkboxes are top aligned
See the code source for examples.

Remark about sizes Please note that when using the euiLabel with the label tag only the default M(edium) and S(mall) sizes are available.
For other custom sizes, please use the eui-u-f-%x utility classes.
