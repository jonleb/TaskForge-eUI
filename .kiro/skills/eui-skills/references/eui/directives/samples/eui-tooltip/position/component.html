With position=%value

where %value is one of the following positions' values :

Position value Spot Description
above Top Always display above the element (default)
below Bottom Always display beneath the element
left Left Always display to the left of the element
right Right Always display to the right of the element
before Left / Right Display to the left in left-to-right layout and to the right in right-to-left layout
after Right / Left Display to the right in left-to-right layout and to the left in right-to-left layout
