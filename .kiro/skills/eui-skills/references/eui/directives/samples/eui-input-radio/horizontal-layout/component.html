Remark about horizontal vs vertical alignements By default, the layout display of an input radio + label is horizontal (inline). In order to have multiple elements displayed horizontally, a flexbox div wrapper should be used around each input + label elements by using any flex container utility class:
  • eui-inline-flex wrapper: is usually used for horizontal layout, gathering the input + label together.
  • eui-u-flex / eui-u-display-flex wrappers: are usually used for vertical layout and for the parent main wrapper.
Spacings (paddings & margins) can also be controlled through the eUI utility classes and from Flexbox gap utility classes. See eUI Flex utility classes.
Recommended spacing size is S(mall). Recommended usage is to use the eui-u-flex-gap-s utility class at parent flexbox wrapper in order to preserve both horizontal and vertical spacings. See the code sources of the following samples for more information.
