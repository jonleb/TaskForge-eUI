Virtual scroll will be activated when there are more than 800 items in the first level of tree array.

It will add to scroll bar to the view, and It will check If It is scrolled to the bottom or top of the container, and will react.

You can see the max-height is defined in order to enable scroll bar at specific height.

This behavior can be customized through the following options:

- virtualScrollThreshold (default: 800) – the minimum number of nodes required before virtual scrolling is enabled.
- virtualScrollPageSize (default: 400) – the number of nodes rendered per page when virtual scrolling is active.
