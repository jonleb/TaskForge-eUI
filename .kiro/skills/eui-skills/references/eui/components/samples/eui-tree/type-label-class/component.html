Badges (on the left side of the node label) can be added to a tree node element by using the following properties (script part).
  • typeLabel : is the label of the badge.
  • typeClass : is the color type which can be one of: primary | secondary | info | success | warning | danger | accent.
Example:
    node: {
        treeContentBlock: {
            label: 'Node 1',
            typeLabel: 'badge',
            typeClass: 'primary',
        },
    },
