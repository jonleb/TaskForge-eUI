Please note that you have to import EuiTreeFormControlDirectiveModule from eui-tree. Then You'll be able to define euiTreeFormControl directive on eui-tree level.

import {{'{'}}EuiTreeFormControlDirectiveModule{{'}'}} from {{'@eui/components/eui-tree'}}{{';'}}

When euiTreeFormControl directive is combined with [formControl] (or formControlName) it will write on the control's model.

customControlValueSetter is there to map chosen items to model. If you don't define it, model control will be populated with tree items. Check the code to see how to map the IDs as model. [id1,id2..]

customControlModelMapper is there to map model to ids in order to render initial selection.
