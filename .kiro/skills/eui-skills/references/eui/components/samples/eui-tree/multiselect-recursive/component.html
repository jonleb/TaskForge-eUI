Tree recursive multiselect feature is available by using both the isMultiselect and isRecursiveSelection input directives. The (selectionChange) event allows to manipulate the tree items when the selection changes (select / unselect a tree item).

Please note that by default, the isRecursiveSelection option also selects the parent node when all the child items are selected. To override the default behaviour, see sample 2 hereafter.
