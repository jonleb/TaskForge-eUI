The eui-tree accepts all sets of eUI Svg icons. For eUI flag icons usage, please refer to the Misc advanced custom layout example.

Usage :

  • iconSvgName : is the Svg icon name (using the default set=eui) or can be a combination of icon name and icon set, separated by a colon (icon:set).
  • iconTypeClass : can be one of primary | secondary | info | success | warning | danger | accent or any color token documented here.
Example:
    node: {
        treeContentBlock: {
            label: 'Notifications',
            iconSvgName: 'eui-notifications',
            iconTypeClass: 'primary',
        },
    },
