Tree single select feature is available by using the isSingleSelect input (When you define isSingleSelect on inputs, it will update all the nodes of your meta-data on runtime as selectable:true as long as you didn't define in meta-data) and will apply selectConfig : {{'{'}}singleSelect :true {{'}'}}. Coupled with the (selectionChange) event it allows to manipulate tree items when the selection changes (select / unselect a tree item).
