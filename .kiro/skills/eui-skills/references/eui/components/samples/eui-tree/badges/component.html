Badges (on the right side of the node label) can be added to a tree node element by using the following properties (script part). An array of badges part of the rightContent section :
  • label : is the label of the badge.
  • typeClass : is the color type which can be one of: primary | secondary | info | success | warning | danger | accent.
Example:
    node: {
        treeContentBlock: {
            label: 'Node 1.6',
            rightContent: {
                badges: [
                    {
                        label: '8',
                        typeClass: 'warning',
                    },
                    {
                        label: '13',
                        typeClass: 'accent',
                    },
                ],
            },
        },
    },
