Tree multiselect feature is available by using the isMultiselect input (When you define isMultiselect on inputs, it will update all the nodes of your meta-data on runtime as selectable:true). Coupled with the (selectionChange) event it allows to manipulate tree items when the selection changes (select / unselect a tree item).
