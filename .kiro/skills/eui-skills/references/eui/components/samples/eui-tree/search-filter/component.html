Custom filter key
If you need to target another property in your content, you can use filterKey parameter of filterTerm function. {{'.filterTerm(val,\'custom\');'}} then If you add {{'{custom: string }'} property to your EuiTreeContentBlockModel, It will check the search text against your custom property. Here we defined '121','122','123' for 'Belgium','Benelux','Europe' When you search for '123' It will show Europe. You can also create some custom properties like '121-Belgium' in order to combine the results for both ID & Label.

Showing children of the matched items
When recursive selection is activated, If user filters, parent item will try to select all of the rendered children. When there is no children, the selection will not fire because of its recursive nature. (if there is no selected children It will be not selected). If you pass showChildrenOfMatchedItems as true {{'.filterTerm(val, undefined, true);'}} It will be rendering children of matched items, to not confuse to the user only matched item will be rendered as toggled, the children will be rendered collapsed.
