This feature requires the import of EuiTemplateDirective
import { EuiTemplateDirective } from '@eui/components/directives';

The isExpanded option can be used to reveal or hide the fieldset contents. By default, the content is displayed (isExpanded=true). Please note that this option only works in conjunction with the isExpandable option. The (expand) output allows to programmatically handle the expandable state of the fieldset.
