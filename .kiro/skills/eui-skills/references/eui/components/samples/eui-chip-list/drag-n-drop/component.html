For Drag and Drop feature, the following samples implement directly CDK Drag and Drop Directives.
This feature needs to be used with template-driven concept.

For horizontal drag and drop, Eui expose utils which allow to make the drag and drop works when multiline.
Horizontal
@for (chip of chipsH; track chip.id) { {{ chip.label }} }
Vertical
@for (chip of chipsV; track chip.id) { {{ chip.label }}
}
Horizontal + between lists
@for (chip of chipsHMulti1; let i = $index; track $index) { {{ chip.label }} }
@for (chip of chipsHMulti2; let i = $index; track $index) { {{ chip.label }} }
Vertical + between lists
@for (chip of chipsVMulti1; let i = $index; track $index) { {{ chip.label }} }
@for (chip of chipsVMulti2; let i = $index; track $index) { {{ chip.label }} }
