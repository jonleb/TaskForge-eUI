Open the browser console to observe events.
Event handler Description
panelClose Triggered when the option panel is closed.
panelOpen Triggered when the option panel is opened.
inputFocus Triggered when the field is focused.
inputBlur Triggered when the field is blured.
clear Triggered when the field is cleared.
selectionChange Triggered when the chips selection is modified.
itemAdd Triggered when a chip is added.
itemRemove Triggered when a chip is removed.
inputChange Triggered when the value of the text field changes.
chipDragStart Triggered when a chip drag starts.
chipDragRelease Triggered when a chip drag is released.
chipDrop Triggered when a chip is dropped.
