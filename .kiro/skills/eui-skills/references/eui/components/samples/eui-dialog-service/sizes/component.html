The eui-dialog uses scalable relative length units for setting the height and width of the Dialog window. Recommended usage is to use the viewport size ie. the browser window size which works like percentages:

  • vh : to set the viewport height
  • vw : to set the viewport width

The hasMobileCustomSize option allows to override the default full screen display when in mobile mode and to set your own height and width sizes.

The isSizeFullScreen option allows to sets automatically the width and height to 100% and overrides the width and height parameters.

The Service driven approach allows to dynamically set the Dialog window size.
Please note that when in mobile mode, the Dialog window is always displayed in full screen by default, unless you use the hasMobileCustomSize=true option which allows to set your own height and width sizes.
