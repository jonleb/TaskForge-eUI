Because EuiDialogService is declared with { providedIn: 'root' }, it is created inside the root injector. This means it only has visibility over the providers available at the same injector level (global providers). It does not have access to the local providers defined inside the components that call the dialog.

When a component is instantiated through a ComponentPortal, Angular does not automatically inherit the caller's providers. To make the injected component see the local providers of the calling component, you must explicitly pass the caller's injector to the dialog service through openDialog(config, injector) method.
