Search string can be highlighted in the results of the table with the pipe: euiTableHighlightFilter.
Example: <span [innerHTML]="row.country | euiTableHighlightFilter : strFilter : 'highlight-class'"></span>
First parameter is the search string of the field, second id the class to be used for highlighting, default is: eui-table--highlighted
