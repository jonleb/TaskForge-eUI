Responsive tables can be obtained by adding the isTableResponsive template directive next to the table tag.
As a general recommendation, responsive tables allows scrolling on smaller devices viewports as well as a more readable layout like "cards" for mobile and tablets.

For doing so, you need to add the data-col-label attribute next to each td with the corresponding column label. See code source for example.

Try to resize your browser to see it in action.
