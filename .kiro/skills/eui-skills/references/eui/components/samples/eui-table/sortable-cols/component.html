Sorting is activated by adding isSortable on a th. Multi-sorting is activated by adding isMultiSortable and isSortable.
Simple sort and multi-sort can be used in the same table.
The property on which the sort will be done needs to be specified in the sortOn option.
Sort changes will emit an event accessible through output sortChange.
