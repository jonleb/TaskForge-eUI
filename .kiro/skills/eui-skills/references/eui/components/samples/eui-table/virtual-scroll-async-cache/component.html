eui-table selector needs to be used in order to make virtual scroll working properly.
The table needs to be inside a container having height defined in px or vw/vh.
To make the virtual scroll working properly, each rows needs to have the same height.
The source code of the used service can be found here.

With isVirtualScrollCache option, the table does trigger scrollChange for the data already loaded.
