The following samples shows how to use and inject, on demand, components inside the Dialog window and make them communicate together.
Injected components can be declared in a Template driven approach or through the EuiDialogService.

Observable can be used to refresh the view.

Note that when using Template driven approach, the OnInit and OnDestroy hooks of the injected components will be triggered with the ones of the parent page. To prevent this behaviour, the below samples work with the following solution:


<eui-dialog #dialog>
    @if (dialog.isOpen) {
        <my-component></my-component>
    }
</eui-dialog>
