You can build up and use you own custom set of SVG icons by using the following procedure.
The recommended starting point is to generate your custom icons set from a fresh eUI-CLI generated app.
Recommended step by step procedure
STEP 1 - Setup
Temporary install "SVGSpriter" package to your root app package.json file dependendies. This package is required in order to generate the Svg sprites. It is not recommended to keep it for environments other than your local dev. Add this line to the dependendies list : "svg-sprite": "2.0.4"
STEP 2 - Sources
Copy your custom svg icons to your app root assets folder in: assets/icons/svg-src
STEP 3 - Generation
Generate your custom Svg icons set file with the following command prompt : 
    npm run generate-sprite -- --inFolder src/assets/icons/svg-src --outFile custom-set.svg

The script will create an assets/icons/sprites folder containing 2 files:

  • custom-set.json : a Json file containing the list of the icons in your set
  • custom-set.svg : custom Svg icons set containing all the sprites
The sprites set will be loaded automatically when serving the App.
The "svg-src" folder is no more required and can be removed after generation.
STEP 4 - Usage
In order to display your new icons with the eui-icon-svg component, you just need to pass the set name.

Example :

<eui-icon-svg icon="eui-logo:custom-set" />
