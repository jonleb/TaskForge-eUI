Important usage notice In order to use the eUI Flag icons library you need to import extra styles dependencies from your project's angular.json config file.
In the "styles" section, add the following stylesheet import :
"node_modules/@eui/styles/dist/styles/eui-icons-flags.css",

Usage samples :

    <div>
        <span class="eui-flag-icon eui-flag-icon-ad"></span>
        <span class="eui-flag-icon eui-flag-icon-ae"></span>
        <span class="eui-flag-icon eui-flag-icon-af"></span>
        <span class="eui-flag-icon eui-flag-icon-ag"></span>
        . . .
    </div>
