You can use any custom SVG icon by providing the URI of the SVG file with the iconUrl input option.
In order to have your custom SVG icon's files accessible for your project, it is recommended to store them within your project's assets folder. Please note that when using custom SVG icons, some options may not work (like the color which is usually embedded within the SVG data file). You can also use SVG sprites to create your own custom set. See "Custom icons sets" section from right menu.

Custom size

You can define a custom size by setting the height and width through the style tag.
Examples :

    <eui-icon-svg iconUrl="assets/images/ec-logo-svg.svg" style="height: auto; width: 200px;" />
    <eui-icon-svg iconUrl="assets/images/ec-logo-svg.svg" style="height: auto; width: 20rem;" />
