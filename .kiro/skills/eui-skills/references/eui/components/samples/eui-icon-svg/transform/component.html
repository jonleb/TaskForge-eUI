Use the transform input option to apply some transformations to the icon. Transform accepts the following values :

rotate-45 | rotate-90 | rotate-135 | rotate-180 | rotate-225 | rotate-270 | rotate-315 | flip-horizontal | flip-vertical | flip-both | stack | spin
transform="rotate-xx"

Default (no transform)

transform="stack"

In order to properly stack 2 Svg icons you need to follow some basic rules:

  • first icon: choose a bigger size than the second one. It will act as the "background" icon.
  • second icon: choose a smaller size than the first icon and add the transform="stack" option
  • If the icons are not perfectly aligned, add a spacing utility class to the transform option. Example: transform="stack eui-u-mr-s" will add some extra margin to the right of the second icon in order to be centered with the first one. Of course, the extra spacing is depending on the size ratio between the 2 icons. See the code source for more information.
