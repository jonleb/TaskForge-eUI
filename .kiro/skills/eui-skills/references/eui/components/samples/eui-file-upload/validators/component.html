The eui-file-upload provides built-in ready to use validators to help user to validate their forms.

Usage : import { maxFilesValidator, maxSizeValidator, mimeTypeExtensionValidator } from '@eui/components/eui-file-upload';

Validator Description
maxFilesValidator(number) Validator that requires the maximum length of files that can be uploaded.
maxSizeValidator(number) Validator that requires the maximum size (in bytes) of files that can be uploaded.
maxFileSizeValidator(number) Validator that requires the maximum size (in bytes) per file that can be uploaded.
mimeTypeExtensionValidator(mimeType[]) Validator that requires an array of mime type that can be uploaded.
