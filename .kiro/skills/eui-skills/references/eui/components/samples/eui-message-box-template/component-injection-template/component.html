The following samples shows how to use and inject, on demand, components inside the Message Box window and make them communicate together. Injected components can be declared in a Template driven approach or through the EuiMessageBoxService.

Note that when using Template driven approach, the OnInit hook is triggered when the page is loaded and OnDestroy hook when the page has changed. If you need 'ngOnDestroy' logic, it needs to be written in the close callback of the Message Box.
