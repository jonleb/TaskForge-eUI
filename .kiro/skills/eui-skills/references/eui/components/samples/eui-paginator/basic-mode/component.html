isBasicMode

The isBasicMode input option allows to display a simple paginator which is typically used when working with async data fetching or where the page sizes and/or the number of total records are not set.

hasPageSize

When the page size of the fetched data cannot be provided (or the total of the number of records is unknown), you can hide the display of the paginator's left section by using the isBasicMode hasPageSize=false input option (default: true).
