Page numbers allows more precise and fast navigation through data pages which also brings a better user experience when displaying lots of data records.
Use the hasPageNumberNavigation=true input property to enable the display of pages numbers to be accessible within the next and previous action buttons. Default number of accessible pages : 5.

Combining the page number navigation with the nbPageNumberNavigation input property, you can define the number of numbered pages to be accessible. Recommended and default value: 5
