Multi-lines header

By default, the card header's title and subtitle are displayed on a single row and automatically truncated (using the text-overflow ellipsis property) when content exceeds the size of the container. Title & subtitle's content can be displayed in multiple lines by using the following option: eui-card-header isHeaderMultilines.

Multi-lines {{ isMulti }}
{{ cardTitle }} {{ cardSubtitle }}

In conjunction with the isHeaderMultilines option, you can also control the number of characters to be displayed by using the euiTruncate pipe. The following example uses the following truncation values :

  • card title: truncate after 100 characters
  • card sub-title: truncate after 255 characters

{{ cardTitle | euiTruncate: 100 }} {{ cardSubtitle | euiTruncate: 255 }}
