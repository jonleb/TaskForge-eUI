This feature requires the import of EuiTemplateDirective
import { EuiTemplateDirective } from '@eui/components/directives';

Uses the eui-card as the main container including :

  • the card header section : displays a custom title and actions (search filter, edit mode)
  • the card content section : displays the eui-table using a card's list as table rows

Using eui-table component as part or the eui-card-content template directive.
It also uses the euiNoContentPadding option to fill the main card content container.
