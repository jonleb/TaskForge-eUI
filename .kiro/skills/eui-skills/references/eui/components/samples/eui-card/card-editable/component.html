All eui-card elements can be editable: header title/subtitle, header body content and main content. The following sample uses the eui-editor Rich Html content editor for editing the card main content. The editor is populated using reactive forms.

Edition of cards can be triggered within the card itself like the following example or at page level, for global or context related edition.

To activate the edit mode of an eui-card, use the euiHighlighted=true input directive. This will dimmer the screen and reveals (highlights) only the editable elements.

Usage : 
    <eui-card [euiHighlighted]="isEditActive">

See the code source for more information.
