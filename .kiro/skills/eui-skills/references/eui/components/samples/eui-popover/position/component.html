Popover initial position can be defined by the position input option which accepts the following values: top | right | bottom (default) | left


Important remark about automatic repositioning

When the parent container's element is too small to display the popover contents at the choosen potition, a fallback is automatically provided to best re-position the popover overlay on the screen.
