The width of the dropdown panel can be set using the width input option. Value needs to be one of the following measurement units: px, vw or rem. Default value: auto

Default size width is set to auto which automatically defines the width to best fit the content within its container.
If content is longer than the content container, it will be automatically truncated by ellipsis to fit the container.
Please note that for UX reasons, the minimum width is set by default to 8rem.

In order to have long labels automatically truncated by ellipsis, the label should be surrounded by the euiLabel directive or by the eui-label tag.
