The following samples shows how to use and inject, on demand, components inside the Message Box window and make them communicate together. Injected components are declared through the EuiMessageBoxService.

The source code of the injected component can be found here.

Inject component in message box body and footer
In the following sample, the component injected in the footer has a selector property then its content is embedded in a tag having this value. When selector property is not provided, Angular creates a ng-component tag instead. eUI sets ng-component width to 100%.

If a selector is needed, a class needs to be added on the host and width sets to 100% or, as shown in the footer, use eui-u-flex utility class: 
    
    @HostBinding('class') classes = 'eui-u-flex';
