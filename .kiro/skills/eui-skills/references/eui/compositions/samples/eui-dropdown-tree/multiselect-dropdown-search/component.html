Responsive multi-select dropdown with a search filter

This advanced composition sample features a responsive dropdown with rich content containing :
- an auto search filter with clearable option
- an expand/collapse toggle icon button with tooltips
- a select/deselect all items option
- the multi-select eui-tree

Please note that the button trigger uses the euiResponsive option which allows the selected tree items to be displayed correctly within the button. Also, the eui-dropdown have the isBlock input option which displays the dropdown content using the same width as the button trigger.
