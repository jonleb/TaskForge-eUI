The isAutocloseOnContainerResize input option combined with the autocloseContainerWidth allows to automatically collapse (close) the eui-page-column and the eui-sidebar-menu when the viewport's width reaches the defined value of autocloseContainerWidth.

For example, if you want to have the first column to be automatically collapsed on tablet/laptop (landscape) mode, use : 
<eui-page-column ... isCollapsible isAutocloseOnContainerResize [autocloseContainerWidth]="1200">
