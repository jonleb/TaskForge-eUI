Column collapse with icons

When using the isCollapsedWithIcons input option combined with the isCollapsed input, components like the eui-sidebar-menu using menu items with icons will show the icons when the column is collapsed (closed).

Please refer to the eui-sidebar-menu component for more information and all available options you can use (hasTooltip, hasCollapsedInitials, etc.).
