Columns can be sized using the size input option. Allowed values ranges from euiSizeS for the smallest size to euiSize4xl for the largest one.

If no size is provided, column adjust proportionally to the width of their container (here the eui-page/eui-page-columns wrapper)

For responsive design, it is recommended to never set a size for the main column which automatically grows based on the page layout and content. If all columns have no defined size value, their sizes will be evenly distributed in proportion to their parent container (eui-page-columns).
