Event handler Description
refreshClick Triggered when the 'refresh' link is clicked.
notificationsClick Triggered when the bell icon is clicked and open the overlay.
viewAllClick Triggered when the 'view all' link is clicked.
settingsClick Triggered when the 'settings' icon is clicked.
markAllAsReadClick Triggered when the 'mark all as read' link is clicked.
noNotificationFoundClick Triggered when the 'no notification' link is clicked. (in combination with noNotificationFoundLink = true)
itemClick Triggered when a notification is clicked.
itemMarkAsReadClick Triggered when the 'mark as read' link of a notification is clicked.
