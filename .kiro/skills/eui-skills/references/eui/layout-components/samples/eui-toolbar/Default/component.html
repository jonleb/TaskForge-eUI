In order for keyboard navigation to work on the user profile menu, make sure that you project directly the eui-user-profile-menu-item components without using *ngTemplateOutlet. When EuiUserProfileMenuItemComponent is instantiated by *ngTemplateOutlet from the userProfileMenuItems, templates are not considered direct content children of eui-user-profile-menu in the way @ContentChildren expects and thus the first item is not focused. More info can be found here: Angular issue #14842.
