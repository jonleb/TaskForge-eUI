The eui-overlay panel layout can be fully customized by using 3 different sections: the header, the body and the footer.

Use the following tags for each specific section :
  • eui-overlay-header : displays the header of the overlay panel. Usually contains the overlay title and the close overlay's action button.
  • eui-overlay-body : displays the scrollable main body content (text, forms, lists, etc.)
  • eui-overlay-footer : displays the optional footer of the overlay panel. Usually gathers actions buttons.
