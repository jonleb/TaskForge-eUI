The eui-overlay-header section layout can be controlled automatically by using the eui-overlay-header-title tag.

It allows 3 input options :
  • headerTitleLabel : displays the header title label
  • headerTitleHideLabel : displays the close/hide overlay panel trigger label
  • headerTitleCount : optional, displays a badge counter next to the title.
