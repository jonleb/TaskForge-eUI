The eui-overlay panel has default width sizes based on the device's screen width (Media Queries).

Default overlay panel widths size :
  • mobile : 75% of the screen
  • tablet : 45% of the screen
  • desktop : 33% of the screen

You can also use predefined width sizes by using the width input option and choose one of the following values:

  • sets the overlay width to 25% of the screen (default)
  • sets the overlay width to 30% of the screen
  • sets the overlay width to 35% of the screen
  • sets the overlay width to 40% of the screen
  • sets the overlay width to 50% of the screen
  • sets the overlay width to 75% of the screen
Sample code:
<eui-overlay [isActive]="{{ isOverlayActive }}" [width]="{{ overlayWidth }}"></eui-overlay>

You can also use fixed width sizes by using the fixedWidth input option and choose one of the following values:

  • sets the overlay width to 25rem of the screen (default)
  • sets the overlay width to 30rem of the screen
  • sets the overlay width to 35rem of the screen
  • sets the overlay width to 40rem of the screen
  • sets the overlay width to 50rem of the screen
  • sets the overlay width to 75rem of the screen
Sample code:
<eui-overlay [isActive]="{{ isOverlayActive }}" [fixedWidth]="{{ overlayFixedWidth }}"></eui-overlay>
