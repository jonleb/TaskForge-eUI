When using the isCollapsedWithIcons input option combined with the isCollapsed input, the eui-sidebar-menu with hasCollapsedInitials option, will display the label first 2 initials instead of the icons when the column is collapsed (closed).

Please refer to the eui-sidebar-menu component for more information and all available options you can use (hasIcons, hasTooltip, hasCollapsedInitials, etc.).
