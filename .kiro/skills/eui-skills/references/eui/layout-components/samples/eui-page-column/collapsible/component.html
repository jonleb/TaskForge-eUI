Columns can be collapsed and expanded using the isCollapsible input option. Its state can be set by adding the isCollapsed input option which sets the layout to :

  • closed column (collapsed) : using isCollapsible & isCollapsed=true (default: false)
  • opended column (expanded) : using isCollapsible & isCollapsed=false
Simple column collapse (default)

Column state can also be set programmatically :
